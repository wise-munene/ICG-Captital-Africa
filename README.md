# ICG-Captital-Africa 

#### Date, 2026/09/25

#### By *Wise Munene *

## Description
This is a full-stack web application for ICG Capital Africa, a consulting and investment advisory firm. The platform allows users to explore services, request quotes, and contact the company, while providing an admin dashboard to manage and respond to client requests.

## Installation
You use git clone to be able to download the documents in the GitHub

## Installation Requirements
Git
Node.js & npm
Python(Flask)
PostgresSql

### Installation instruction
```
Git clone https://github.com/hamida-mstafa/Triangle-tracker.git

cd ICG-Capital-Africa
cd backend
pip install -r requirements.txt
flask db upgrade
python run.py

cd frontend
npm install
npm run dev
```
